# FlowReveal - HTTP/HTTPS 流量捕获工具验证清单

## 核心功能验证
- [ ] 检查点 1: 系统级流量捕获 - 能够捕获整个系统范围内的 HTTP/HTTPS 流量，包括本地主机连接
- [ ] 检查点 2: HTTP 协议解析 - 能够正确解析并显示 HTTP 方法、URL、头部、Cookie、状态码等信息
- [ ] 检查点 3: HTTPS 流量解密 - 能够以明文形式显示解密后的 HTTPS 内容，无需手动安装证书
- [ ] 检查点 4: 实时终端输出 - 能够实时将捕获的流量以可读格式输出到终端
- [ ] 检查点 5: 流量过滤 - 能够按域名、HTTP 方法、状态码等条件过滤流量
- [ ] 检查点 6: 会话管理 - 能够关联 HTTP 请求和响应，重建完整的会话
