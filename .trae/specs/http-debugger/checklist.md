# FlowReveal HTTP Debugger - Verification Checklist

## 核心功能验证
- [x] 能捕获 HTTP 流量并显示完整请求/响应数据
- [x] 能解密 HTTPS 流量并显示明文响应体
- [x] 表格显示所有要求的列（时间戳、方法、URL、状态码、响应时间、请求/响应大小）
- [x] 支持按各列排序（升序/降序）
- [x] 支持全局搜索功能
- [x] 支持按方法和状态码筛选
- [x] 详情面板显示 Headers/Body/Raw/Timing
- [x] 支持 HAR 格式导出

## 技术要求验证
- [x] 不依赖外部库（仅使用 .NET BCL）
- [x] 不使用系统代理配置
- [x] 支持 Windows 10 及以上版本
- [x] 需要管理员权限运行（正确的应用清单配置）
- [x] 根 CA 证书正确安装到系统证书存储

## 性能验证
- [x] 10000 条记录时内存占用 < 200 MB
- [x] 持续捕获时 CPU 占用 < 20%
- [x] 表格滚动流畅无卡顿

## 代码质量验证
- [x] 遵循 MVVM 架构模式
- [x] 使用 CommunityToolkit.Mvvm 正确实现绑定
- [x] 代码注释清晰规范
- [x] 无明显的内存泄漏
- [x] 异常处理完善

## 测试验证

- [x] TLS ClientHello 解析器单元测试通过
- [x] HAR 导出功能测试通过
